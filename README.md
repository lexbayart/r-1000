# Р-1000 🔷 [Beta]

> Нейросеть предлагает — вы подтверждаете или отвергаете. Собираете размеченные данные для обучения модели распознавания звука «Р».

Без установки. Без аккаунта. Работает офлайн.

---

## 🔷 Для чего это

Р-1000 — инструмент сбора размеченных данных для исследований в области логопедии.

Проблема: чтобы нейросеть научилась точно распознавать фонему «Р» в речи, ей нужны тысячи примеров — и правильных, и неправильных. Собирать эти примеры вручную, прослушивая часы видео, неэффективно.

Решение: модель Edge Impulse делает первый проход по звуковой дорожке и выделяет участки, где, по её мнению, произносится «Р». Вы прослушиваете каждый момент и ставите ✅ (да, это Р) или ❌ (нет, не Р). Полученные позитивные и негативные примеры скачиваются архивом — и идут на дообучение модели.

Каждый новый цикл: модель предлагает → вы размечаете → модель учится → становится точнее.

## ✨ Возможности

- **🧠 Нейросетевой поиск «Р»** — модель Edge Impulse сканирует звуковую дорожку каждого видео и автоматически выделяет участки с вероятным произношением
- **✅ / ❌ Разметка** — подтверждаете правильные примеры и отвергаете ложные срабатывания
- **📊 Визуализация звуковой дорожки** — волна с интерактивной шкалой: видите, где модель нашла «Р», перемещаетесь по дорожке, выделяете окна
- **🎛️ Настройки модели** — порог уверенности, шаг анализа, масштаб сигнала, нормализация громкости — 4 пресета + ручная подстройка каждого параметра
- **📦 Экспорт данных** — скачиваете собранные позитивные и негативные примеры отдельно, архивом, для дообучения модели
- **💾 Сохранение прогресса в файле** — разметка вшивается прямо в видеофайл; при повторном открытии ваши метки восстанавливаются
- **🎬 Коллаж видео** — загружайте сразу несколько роликов, переключайтесь между ними, перетаскивайте плитки
- **📁 Загрузка папки** — один диалог на всю папку, дальше без вопросов
