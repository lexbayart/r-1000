# Р · нарезка 🔷 [Beta]

> Видеоредктор, который сам находит «Р» в звуковой дорожке с помощью нейросети — а вы проверяете, правильно ли она угадала.

**[▶ Открыть в браузере — попробовать сейчас](https://lexbayart.github.io/r-sound-editor/)**
Без установки. Без аккаунта. Работает офлайн после первой загрузки.

---

## 🔷 Что делает

Обычные видеоредкторы требуют вручную искать и вырезать каждый момент. Здесь нейросеть Edge Impulse сама прослушивает аудио каждого видео и подсвечивает на волне участки, похожие на звук «Р». Вы перематываете к каждому такому моменту, слушаете и ставите ✅ (правильно) или ❌ (неправильно).

Это не транскрибация и не распознавание речи — это точечный детектор одного конкретного звука. Модель обучена именно на «Р» и не пытается понять слова.

Метки вшиваются прямо в файл видео (как скрытый MP4-бокс). Если закрыть вкладку и открыть тот же файл снова — предыдущая разметка восстановится автоматически.

---

## ✨ Возможности

- 🎬 **Коллаж видео** — перетащите несколько файлов, они расставятся плитками на весь экран
- 🧠 **Нейросетевой поиск «Р»** — Edge Impulse модель (WebAssembly) сканирует аудио и подсвечивает зоны на волне
- 📊 **Волна с разметкой** — зелёные/красные зоны, линейка времени, курсор воспроизведения
- ✅❌ **Ручная проверка** — слушаете каждый момент и подтверждаете или отвергаете
- 📦 **Экспорт архивами** — скачиваете «правильные» и «неправильные» нарезки отдельными zip-файлами
- 💾 **Метаданные в файле** — разметка вшивается в исходный MP4, восстанавливается при повторном открытии
- 🎛 **Настройки нейросети** — порог, интервал, масштаб, нормализация громкости, 4 пресета
- ⌨️ **Горячие клавиши** — всё управление с клавиатуры: пробел, стрелки, Enter, Backspace, Tab, P
- 📁 **Загрузка папки** — один диалог на всю папку, все видео загружаются сразу
- 🔍 **Индикатор сканирования** — прогресс-бар с процентом и ETA при анализе дорожки
